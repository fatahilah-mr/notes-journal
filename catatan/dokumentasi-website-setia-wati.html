🏗️ Arsitektur Komponen (App.tsx)

Proyek ini dibangun sebagai Single Page Application (SPA) dengan satu komponen utama App.tsx yang menangani seluruh logika dan tampilan. Berikut rincian arsitektur internalnya:

State Management (Hooks React)

  • isMenuOpen - mengontrol status buka/tutup menu mobile (drawer).
  • activeSection - melacak bagian halaman yang sedang aktif (scroll spy) untuk menyorot tautan navbar.
  • isModalOpen - status tampilan modal formulir kontak (email).
  • likedTracks - array indeks lagu favorit (default: semua lagu disukai). Pengguna dapat toggle like pada setiap lagu di playlist.

Fungsi & Event Handlers

  • handleSendEmail() - membuka mail client default dengan subjek dan isi pesan yang telah diisi user (menggunakan protokol mailto:).
  • handleSmoothScroll() - navigasi halus antar-section dengan offset navbar (85px) untuk menghindari tertutup.
  • handleMouseMoveTilt() - efek tilt 3D pada kartu idola saat kursor bergerak (menggunakan transformasi CSS rotateX/Y).
  • handleMouseLeaveTilt() - mengembalikan kartu ke posisi normal saat kursor meninggalkan area.

Effect (useEffect)

  • Scroll Spy: Menggunakan useEffect dengan event listener scroll (passive) untuk memperbarui activeSection sesuai bagian halaman yang sedang dilihat.

🔍 Optimasi SEO & Deployment Cloudflare

1. Meta Tag di index.html

<meta name="description" content="Setia Wati (Kwettiau) - Lulusan Teknik Komputer dan Jaringan (TKJ) SMK Patriot Pituruh berusia 18 tahun. Temukan petualangan hidup saya di bidang teknologi, karya tulis, dan kreasi kuliner." />

2. robots.txt

User-agent: *
Allow: /

Sitemap: https://kwettiau.fatahmr.my.id/sitemap.xml

3. sitemap.xml

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://kwettiau.fatahmr.my.id/</loc>
    <lastmod>2026-06-30</lastmod>
    <changefreq>monthly</changefreq>
    <priority>1.0</priority>
  </url>
</urlset>

4. Penanganan SPA Redirect (_redirects)

Mencegah error 404 saat refresh di rute selain / dengan aturan:

/*    /index.html   200

📱 Prinsip Mobile-First

Seluruh proses pengembangan-mulai dari instalasi dependensi, penulisan kode, debugging, hingga deployment-dikelola sepenuhnya dari perangkat seluler (HP) tanpa komputer desktop. Ini membuktikan fleksibilitas alur kerja modern dan efisiensi prompt engineering tingkat lanjut.

📝 Catatan Perbaikan Terakhir

  • Penambahan file robots.txt, robot.txt, dan sitemap.xml untuk optimasi Google Search Console.
  • Pembuatan _redirects untuk mencegah error 404 pada SPA di Cloudflare Pages.
  • Navbar fixed (tanpa auto-hide) untuk kemudahan navigasi.
  • Validasi build dengan tsc --noEmit dan npm run build - status sukses 100%.
