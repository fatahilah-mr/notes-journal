📌 Spesifikasi Umum Proyek

Portofolio digital ini dirancang sebagai platform pameran kredensial profesional dan galeri menu kuliner milik Arifin Prasetyo. Target utama dari platform ini adalah pemilik bisnis kuliner, hotel bintang, katering berskala besar, serta rekruter industri Food & Beverage (F&B).

Spesifikasi Lingkungan Produksi

  • Domain Utama: arifin.fatahmr.my.id
  • Infrastruktur Hosting: Cloudflare Pages (Edge Network)
  • Runtime Environment (Lokal): Bun (JavaScript/TypeScript Runtime)
  • Metode Rendering: Client-Side Rendering (CSR) dengan optimasi perayapan bot.

🏗️ Arsitektur Kode & Konvensi Penamaan

Proyek ini mengadopsi standar arsitektur berbasis komponen modular dengan pemisahan fungsi yang tegas (Separation of Concerns).

TanStack File-Based Routing

Alih-alih menggunakan sistem routing tradisional berbasis komponen (react-router-dom), proyek ini menerapkan TanStack Router. Keuntungannya meliputi:

  • Type-Safety Mutlak: Memastikan semua rute dan parameter terdeteksi aman sejak masa kompilasi (compile-time).
  • Struktur Berbasis File: Struktur folder dalam src/routes/ mencerminkan hierarki URL secara otomatis:
    • src/routes/__root.tsx: Berfungsi sebagai layout utama pembungkus aplikasi (Navbar, Footer, global state).
    • src/routes/index.tsx: Berfungsi sebagai halaman utama (Landing Page) tempat seluruh seksi komponen dimuat.

Konvensi Kode

  • Komponen UI (src/components/): Ditulis menggunakan ekstensi .tsx dengan konvensi penamaan berkas PascalCase (Contoh: PortfolioGrid.tsx, Certifications.tsx).
  • Styling (src/styles.css): Menggunakan Tailwind CSS untuk utilitas kelas responsif penuh, dipadukan dengan konfigurasi warna tema fine-dining khusus (Deep Maroon #8B0000 dan Elegant Gold #D4AF37).
  • Aset Gambar (src/assets/): Semua gambar bersertifikat dan foto menu dikonversi ke dalam format modern .webp untuk memangkas ukuran berkas tanpa merusak kualitas visual demi kecepatan muat di perangkat ponsel pintar.

⚙️ Logika Komponen Internal (Galeri & Filter)

Komponen PortfolioGrid.tsx menangani klasifikasi 14 jenis hidangan secara dinamis menggunakan manajemen state internal React tanpa memicu pemuatan ulang halaman (Zero-Reload).

// Ilustrasi Logika Filter Kategori pada Komponen PortfolioGrid
const [activeCategory, setActiveCategory] = useState<string>("Semua Menu");

const filteredMenus = activeCategory === "Semua Menu"
  ? totalMenus
  : totalMenus.filter(menu => menu.category === activeCategory);

UX Responsif Perangkat Mobile

Untuk memastikan kenyamanan akses sepenuhnya dari layar ponsel pintar:

  • Baris tombol filter kategori menggunakan pembungkus dengan aturan CSS overflow-x: auto; white-space: nowrap; yang mengizinkan interaksi usap (swipe) horizontal yang halus.
  • Grid kartu menu otomatis beralih menjadi struktur satu kolom penuh pada layar berukuran kecil (HP), dan memisahkan diri menjadi multi-kolom secara proporsional saat mendeteksi layar desktop.

🚀 Analisis Optimasi SEO & Deployment Cloudflare

Sebagai aplikasi satu halaman (SPA), proyek ini rentan terhadap kendala indeksasi mesin pencari dan penanganan rute rujukan balik (deep linking). Solusi teknis berikut diimplementasikan langsung pada folder public/:

Penanganan Error 404 pada Cloudflare Pages via _redirects

Secara fisik, server hanya mengenali file index.html. Jika rekruter melakukan penyegaran halaman (refresh) di luar rute utama, server Cloudflare akan melempar error 404. Berkas _redirects menginstruksikan server untuk memetakan seluruh rute kembali ke index.html dengan status sukses 200 OK:

/*    /index.html   200

Strategi Perayapan Google Search Console

Aset JavaScript hasil kompilasi Vite diloloskan agar dapat dibaca sempurna oleh Googlebot melalui instruksi khusus pada robots.txt:

User-agent: *
Allow: /
Allow: /assets/

Sitemap: https://arifin.fatahmr.my.id/sitemap.xml

Peta situs dikirimkan dalam format XML terstruktur tunggal (sitemap.xml) yang merujuk pada domain produksi utama dengan prioritas indeksasi penuh (1.0).
