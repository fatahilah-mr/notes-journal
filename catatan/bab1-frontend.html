1. HTML5 (Semantic Markup)

Definisi: Penyusunan struktur halaman web menggunakan elemen yang memiliki arti logis, bukan sekadar menggunakan tag generik seperti <div>.

Fungsi: Mempermudah mesin pencari (SEO) mengenali struktur konten dan mempermudah pemeliharaan kode (maintainability) dalam jangka panjang.

Elemen yang Dikuasai:

  • <header>: Bagian kepala atau pengantar halaman (judul dashboard/nama profil).
  • <main>: Area konten utama yang menjadi pusat informasi halaman.
  • <section>: Pembagi konten menjadi beberapa kluster/bagian logis (misal: bagian navigasi tombol dan bagian penampilan materi).
  • Atribut id: Penanda unik pada elemen HTML agar bisa ditembak dengan akurat oleh CSS dan JavaScript.

2. CSS3 (Modern Styling & Responsive Design)

CSS Custom Properties (CSS Variables):

  • Sintaks: Ditulis di dalam lingkup global :root { --nama-variabel: nilai; } dan dipanggil menggunakan var(--nama-variabel).
  • Keuntungan: Efisiensi tinggi. Menghindari penulisan kode warna atau ukuran yang sama secara berulang (hardcode). Cukup mengubah satu baris di :root, maka seluruh tampilan website akan berubah secara instan.

Mobile-First & Media Queries:

  • Pendekatan: Mendesain tampilan dasar yang dioptimalkan untuk layar HP terlebih dahulu.
  • Sintaks: @media (min-width: 768px) { ... }
  • Logika: Instruksi jika-maka (if-statement) pada browser. Jika lebar layar perangkat minimal 768px (tablet/laptop), maka jalankan aturan CSS di dalamnya (seperti pembatasan max-width dan pemosisian ketengah dengan margin: 0 auto). Jika di bawah itu (layar HP), abaikan.

Troubleshooting (Pembersihan Inline Style):

Menghindari penulisan atribut style="..." langsung di dalam tag HTML karena memiliki prioritas lebih tinggi (specificity) daripada file CSS eksternal/internal. Kemarin sempat merusak kontras warna (teks putih menabrak latar abu-abu terang), dan berhasil diatasi dengan menyerahkan kendali visual sepenuhnya ke blok <style> utama.

3. JavaScript Vanilla (Logika & Dinamisasi DOM)

  • DOM Selection: Menggunakan document.getElementById('id-elemen') untuk menghubungkan objek HTML ke dalam memori JavaScript.
  • Event Handling: Menggunakan .addEventListener('click', function() { ... }) untuk mendengarkan dan merespons ketukan pengguna pada layar HP secara real-time.
  • Manipulasi Konten Dinamis:
    • textContent: Digunakan untuk menyuntikkan teks murni dengan aman tanpa risiko celah keamanan.
    • innerHTML: Digunakan ketika teks yang disuntikkan mengandung tag HTML baru (seperti <br> untuk baris baru atau <strong> untuk cetak tebal).
  • Arsitektur Berbasis Data (Data-Driven Content):
    • Konsep: Memisahkan konten/data dari struktur tampilan. Konten disimpan di dalam struktur data Array of Objects di JavaScript.
    • Sintaks Utama: daftarMateri.forEach(function(materi) { ... }) untuk melakukan perulangan otomatis pada data.
    • document.createElement('button') untuk menciptakan elemen tombol baru langsung dari memori JS.
    • appendChild() untuk memasukkan tombol hasil kreasi JS ke dalam wadah HTML secara otomatis.
    • Keuntungan: Jika ingin menambah materi baru, tidak perlu menyentuh struktur HTML lagi, cukup tambahkan satu baris objek data baru di JavaScript.
